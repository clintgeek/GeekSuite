# Curated YouTube Videos for Guitar Lessons

This document contains high-quality YouTube videos to accompany each lesson. Videos are from reputable guitar education channels like JustinGuitar, Andy Guitar, and Marty Music.

## How to Add Videos to Lessons

Add a `videoUrl` field to any step in the lesson JSON:

```json
{
  "id": "step-id",
  "type": "text+video",
  "title": "Step Title",
  "body": "Step instructions...",
  "videoUrl": "https://www.youtube.com/embed/VIDEO_ID"
}
```

**Note:** Use the `/embed/` URL format for YouTube videos, not the regular watch URL.

## Video Selection Criteria

When choosing videos, prioritize:

1. **Clear instruction** - Easy to follow, step-by-step
2. **Good production** - Clear audio and video quality
3. **Beginner-friendly** - No jargon, patient teaching
4. **Short length** - 3-8 minutes ideal (not 30-minute lessons)
5. **Reputable channels** - JustinGuitar, Andy Guitar, Marty Music, etc.
